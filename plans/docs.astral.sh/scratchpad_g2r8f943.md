# Astral Documentation Analysis

## Main Projects
- **uv**: An extremely fast Python package manager, written in Rust.
  - Root URL: `https://docs.astral.sh/uv/`
- **Ruff**: An extremely fast Python linter and formatter, written in Rust.
  - Root URL: `https://docs.astral.sh/ruff/`

## Observations
- The landing page `https://docs.astral.sh` acts as a portal for these two projects.
- "Get started" button on the portal links to Ruff documentation.
- GitHub Organization: `https://github.com/astral-sh`
- Both project documentations are built with **Material for MkDocs**.
